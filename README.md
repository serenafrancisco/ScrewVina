# ScrewVina
## Automated ensemble docking and virtual screening using AutoDock-Vina

ScrewVina leverages the most popular open-source docking tool to perform molecular docking on multiple receptors or receptor conformations (**ensemble docking**). 

### Features:

🧩**Highly versatile**: you can test as many compounds as you want, against as many receptor structures as you please.

🔓**Easy to access**: extremely simplified code architecture and syntax for everybody.

🎨**Customizable**: configuration files can be easily modified starting from the included example.

ScrewVina was developed at **MedChemBeyond Lab**, University of Torino (Italy).
